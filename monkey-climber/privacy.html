1. 개요

"Monkey Climber"(이하 "앱")는 원숭이가 끝없이 나무를 오르는 캐주얼 무한 클라이밍 게임입니다. 앱은 게임 진행에 필요한 최소한의 정보만 다루며, 글로벌 리더보드·진행도 클라우드 저장·친구 기능을 제공하기 위해 Google Firebase를 사용합니다. 본 방침은 앱이 어떤 정보를 수집·이용하는지 설명합니다.

2. 수집하는 정보

2-1. 사용자가 직접 입력하는 정보

정보수집 방법용도
닉네임리더보드 등록 시 직접 입력리더보드·친구 목록에 표시
친구 코드친구 추가 시 직접 입력친구 연결·선물

2-2. 게임 진행 데이터

  • 점수, 도달 높이, 코인·재화, 업적, 설정값 등 게임 진행 데이터.
  • 기기 내 SharedPreferences에 저장되며, 클라우드 저장·리더보드를 켠 경우 일부 데이터(점수, 닉네임, 진행도)가 Firebase에 저장됩니다.

2-3. 익명 식별자

  • 리더보드·클라우드 저장을 위해 Firebase 익명 인증으로 자동 생성되는 익명 사용자 ID를 사용합니다. 이메일·전화번호 등 계정 정보는 요구하지 않습니다.

2-4. 광고 식별자 (광고 표시 시)

  • 앱은 보상형 광고를 위해 Google AdMob을 사용합니다. 광고 제공 과정에서 Google이 광고 식별자(AAID)와 기기 정보를 사용할 수 있습니다.
  • 앱 내 "광고 제거" 구매 시 광고가 표시되지 않습니다.

2-5. 자동으로 수집되는 익명 오류 정보

앱이 예기치 않게 종료되거나 오류가 발생할 경우, 문제 해결을 위해 아래 익명 오류 정보가 Firebase Crashlytics (Google LLC)로 전송됩니다.

  • 기기 모델명 및 OS 버전
  • 오류가 발생한 코드 위치 (스택 트레이스)
  • 앱 버전
  • 익명의 설치 식별자 (Firebase Installation ID)

이 정보는 특정 개인을 식별하는 데 사용되지 않습니다.

3. 정보의 이용 목적

  • 글로벌·친구 리더보드 순위 표시
  • 게임 진행 데이터의 클라우드 저장 및 복원
  • 친구 시스템(친구 코드 기반 연결·에너지 선물)
  • 보상형 광고 제공
  • 인앱 구매 처리
  • 앱 오류 진단 및 품질 개선

5. 데이터 보관 및 삭제

  • 게임 데이터는 기기(SharedPreferences)와 Firebase Firestore에 저장됩니다.
  • 앱을 삭제하면 기기 내 로컬 데이터는 즉시 삭제됩니다.
  • 앱 내 설정에서 데이터를 초기화할 수 있습니다.
  • 서버에 저장된 데이터(닉네임·점수·진행도)의 삭제를 원하시면 아래 문의 이메일로 요청해 주세요. 요청 시 30일 이내에 삭제합니다.

8. 사용자의 권리 · 옵트아웃

  • 데이터 삭제: 앱을 기기에서 삭제하면 모든 로컬 데이터가 함께 삭제됩니다. 서버 데이터 삭제는 문의 이메일로 요청해 주세요.
  • 광고 맞춤설정 옵트아웃: 안드로이드 기기 설정 → Google → 광고에서 "광고 ID 삭제" 또는 "광고 개인 맞춤설정 사용 중지"를 사용할 수 있습니다.
  • 광고 제거: 앱 내 "광고 제거"를 구매하면 광고가 표시되지 않습니다.

9. 개인정보처리방침 변경

중요한 변경(새로운 데이터 수집 등)이 있을 경우 앱 내에서도 별도 안내합니다.
