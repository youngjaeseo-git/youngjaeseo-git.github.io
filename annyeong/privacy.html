개인정보처리방침 — 안녕, 오늘 안녕, 오늘 · 개인정보처리방침 개인정보처리방침 앱: 안녕, 오늘 (Annyeong, Today) 시행일: 2026년 6월 15일 개발자: 서영재 (youngjaeseo) 문의: youngjae.seo@gmail.com 🔒 일기·기분 기록 등 모든 개인 콘텐츠는 이 기기 안에서만 처리되며, 개발자 서버로 전송되지 않습니다. 1. 개요 "안녕, 오늘"은 매일 아침 현재 위치의 날씨를 귀여운 일러스트와 함께 브리핑하고, 저녁엔 한 줄 기분을 기록받아 캐릭터가 응원/위로를 건네는 앱입니다. 캐릭터의 응원·위로 메시지는 모두 앱 내부 로컬 룰로 생성되며, 사용자가 적은 메모를 외부 AI 서비스로 보내지 않습니다. 2. 수집하는 정보 2-1. 사용자가 직접 입력하는 정보 (기기에만 저장) 정보 수집 방법 저장 위치 닉네임 설정에서 직접 입력 기기 내 생년월일 운세 기능을 켤 때만 직접 입력 기기 내 알림 시간 아침 브리핑·저녁 기분 알림 시간 설정 기기 내 저장한 장소 "내 장소"에 집·회사 등 직접 등록 기기 내 여행 일정 여행 모드에서 직접 입력 기기 내 사용자 추가 명언 설정에서 직접 입력 기기 내 배경 사진 갤러리에서 직접 선택 (선택 사항, 앱 폴더에 복사 저장) 기기 내 캐릭터·음성 설정 설정에서 직접 선택 기기 내 저녁 기분 일기 (기분·태그·메모·캐릭터 응답·그날 날씨 스냅샷) 일기 탭에서 직접 입력 기기 내 저장 방식: Android의 SharedPreferences (앱 데이터 영역). 앱을 삭제하면 모든 데이터가 함께 삭제됩니다. 2-2. 위치 정보 (앱 사용 중에만) * 현재 위치의 날씨를 보여주기 위해 앱 사용 중에만 위치 정보를 요청합니다. * 위치 좌표는 장기 저장하지 않으며, 매번 사용 시 새로 조회합니다. (마지막 조회 위치는 최대 15분간 메모리에만 캐시하여 화면 전환 시 네트워크 부하를 줄입니다.) * 위치 좌표는 Open-Meteo 날씨 API에 전송되어 날씨 정보를 받아오는 데만 사용됩니다. 2-3. 자동으로 수집되는 익명 오류 정보 앱이 예기치 않게 종료되거나 오류가 발생할 경우, 문제 해결을 위해 아래 익명 오류 정보가 Firebase Crashlytics (Google LLC)로 전송됩니다. * 기기 모델명 및 Android 버전 * 오류가 발생한 코드 위치 (스택 트레이스) * 앱 버전 * 익명의 설치 식별자 (Firebase Installation ID) — 광고 식별자(AAID)나 연락처 정보가 아닙니다 * 앱 실행 횟수·화면 진입 횟수 등 익명 집계 통계 이 정보는 특정 개인을 식별하는 데 사용될 수 없으며, 일기 본문·메모·닉네임·생년월일·위치 좌표·저장한 장소 이름·여행 일정·사용자 추가 명언은 포함되지 않습니다. 3. 정보 처리 방식 * 일기·기분 메모는 기기 내에서만 처리되며, 외부 AI/네트워크로 전송되지 않습니다. 캐릭터의 응원/위로 메시지는 모두 앱 내부 로컬 룰로 생성됩니다. * 개인 데이터(일기·메모·닉네임 등)는 어떠한 외부 서버로도 전송되지 않습니다. * 광고 식별자, 행동 추적 데이터는 수집하지 않습니다. 4. 앱 권한 사용 목적 권한 사용 목적 위치 (ACCESS_COARSE_LOCATION / ACCESS_FINE_LOCATION) 현재 위치의 날씨 조회 (앱 사용 중에만) 알림 (POST_NOTIFICATIONS) 아침 브리핑·저녁 기분 일기·여행 출발 전날 안내 (로컬 알림, 서버 없음) 정확한 알람 (SCHEDULE_EXACT_ALARM) Android 12+에서 정해진 시간에 알람 발화 (기상 알람 스타일 선택 시) 풀스크린 알림 (USE_FULL_SCREEN_INTENT) 잠금 화면을 덮는 알람 표시 (기상 알람 스타일 선택 시) 부팅 완료 (RECEIVE_BOOT_COMPLETED) 재부팅 후 예약된 알림 자동 복구 갤러리 사진 (READ_MEDIA_IMAGES) 배경 사진 선택 (선택 사항) 인터넷 Open-Meteo 날씨 조회, 익명 오류 보고 네트워크 상태 오프라인 시 안내 처리 5. 데이터 보관, 삭제 및 자동 백업 보관 및 삭제 * 앱을 삭제하면 기기 내 모든 데이터(일기·설정 등)가 영구 삭제됩니다. * 일기는 마음 달력에서 개별 삭제할 수 있습니다. * 저장한 장소, 사용자 추가 명언 등은 설정에서 개별 삭제할 수 있습니다. * 일기 백업/복원 기능은 AES-GCM-256 + PBKDF2 (10만 회) 암호화 파일로 사용자가 직접 생성·이동하며, 개발자 서버를 거치지 않습니다. Android 자동 백업 앱은 Android의 자동 백업 기능(Android Auto Backup)을 사용합니다. Wi-Fi 연결 및 충전 중일 때, 사용자 본인의 Google 계정 드라이브에 일기·설정 데이터가 자동으로 백업됩니다. * 이 백업은 개발자가 아닌 사용자 본인의 Google 계정에 저장되며, 개발자는 이 데이터에 접근할 수 없습니다. * Google 백업 정책: https://policies.google.com/privacy 6. 제3자 서비스 서비스 제공사 용도 전송 데이터 Open-Meteo Forecast API Open-Meteo 날씨 조회 위치 좌표 Open-Meteo Air Quality API Open-Meteo 미세먼지·꽃가루 조회 위치 좌표 Firebase Crashlytics / Analytics for Firebase Google LLC 익명 오류·집계 분석 기기 정보, 오류 위치, 익명 식별자 Android Auto Backup Google LLC 사용자 데이터 백업 사용자 Google 계정에 직접 저장 관련 정책: * Open-Meteo 약관: https://open-meteo.com/en/terms * Firebase Crashlytics 개인정보처리방침: https://firebase.google.com/support/privacy * Google 개인정보처리방침: https://policies.google.com/privacy 7. 아동 개인정보 만 14세 미만 어린이를 대상으로 하지 않으며, 어린이의 개인정보를 의도적으로 수집하지 않습니다. 8. 사용자의 권리 · 옵트아웃 * 데이터 삭제: 앱을 기기에서 삭제하면 모든 로컬 데이터가 함께 삭제됩니다. * 진단 데이터 옵트아웃: 안드로이드 기기 설정 → Google → 광고에서 "광고 ID 재설정" 또는 "광고 개인 맞춤설정 사용 중지"를 활용하거나, 앱을 삭제하면 됩니다. * 위치 권한 철회: 안드로이드 기기 설정 → 앱 → 안녕, 오늘 → 권한에서 위치 권한을 끄면 즉시 적용됩니다 (이 경우 현재 위치 날씨 기능이 제한됩니다). 9. 개인정보처리방침 변경 이 방침이 변경되는 경우, 본 페이지 상단의 "시행일"을 갱신하고 앱 업데이트 내역에 공지합니다. 중요한 변경(새로운 데이터 수집 등)이 있을 경우 앱 내에서도 별도 안내합니다. 10. 문의 개인정보 처리에 관한 문의는 아래로 연락해 주세요. 이메일: youngjae.seo@gmail.com 본 개인정보처리방침은 대한민국 「개인정보 보호법」 및 Google Play 개발자 정책을 준수하여 작성되었습니다. © 2026 youngjaeseo · youngjae.seo@gmail.com
