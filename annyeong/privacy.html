1. 개요

"안녕, 오늘"은 매일 아침 현재 위치의 날씨를 귀여운 일러스트와 함께 브리핑하고, 저녁엔 한 줄 기분을 기록받아 캐릭터가 응원/위로를 건네는 앱입니다. 캐릭터의 응원·위로 메시지는 모두 앱 내부 로컬 룰로 생성되며, 사용자가 적은 메모를 외부 AI 서비스로 보내지 않습니다.

2-2. 위치 정보 (앱 사용 중에만)

  • 현재 위치의 날씨를 보여주기 위해 앱 사용 중에만 위치 정보를 요청합니다.
  • 위치 좌표는 장기 저장하지 않으며, 매번 사용 시 새로 조회합니다. (마지막 조회 위치는 최대 15분간 메모리에만 캐시하여 화면 전환 시 네트워크 부하를 줄입니다.)
  • 위치 좌표는 Open-Meteo 날씨 API에 전송되어 날씨 정보를 받아오는 데만 사용됩니다.

2-3. 자동으로 수집되는 익명 오류 정보

앱이 예기치 않게 종료되거나 오류가 발생할 경우, 문제 해결을 위해 아래 익명 오류 정보가 Firebase Crashlytics (Google LLC)로 전송됩니다.

  • 기기 모델명 및 Android 버전
  • 오류가 발생한 코드 위치 (스택 트레이스)
  • 앱 버전
  • 익명의 설치 식별자 (Firebase Installation ID) — 광고 식별자(AAID)나 연락처 정보가 아닙니다
  • 앱 실행 횟수·화면 진입 횟수 등 익명 집계 통계

이 정보는 특정 개인을 식별하는 데 사용될 수 없으며, 일기 본문·메모·닉네임·생년월일·위치 좌표·저장한 장소 이름·여행 일정·사용자 추가 명언은 포함되지 않습니다.

3. 정보 처리 방식

  • 일기·기분 메모는 기기 내에서만 처리되며, 외부 AI/네트워크로 전송되지 않습니다. 캐릭터의 응원/위로 메시지는 모두 앱 내부 로컬 룰로 생성됩니다.
  • 개인 데이터(일기·메모·닉네임 등)는 어떠한 외부 서버로도 전송되지 않습니다.
  • 광고 식별자, 행동 추적 데이터는 수집하지 않습니다.

5. 데이터 보관, 삭제 및 자동 백업

보관 및 삭제

  • 앱을 삭제하면 기기 내 모든 데이터(일기·설정 등)가 영구 삭제됩니다.
  • 일기는 마음 달력에서 개별 삭제할 수 있습니다.
  • 저장한 장소, 사용자 추가 명언 등은 설정에서 개별 삭제할 수 있습니다.
  • 일기 백업/복원 기능은 AES-GCM-256 + PBKDF2 (10만 회) 암호화 파일로 사용자가 직접 생성·이동하며, 개발자 서버를 거치지 않습니다.

Android 자동 백업

앱은 Android의 자동 백업 기능(Android Auto Backup)을 사용합니다. Wi-Fi 연결 및 충전 중일 때, 사용자 본인의 Google 계정 드라이브에 일기·설정 데이터가 자동으로 백업됩니다.

  • 이 백업은 개발자가 아닌 사용자 본인의 Google 계정에 저장되며, 개발자는 이 데이터에 접근할 수 없습니다.
  • Google 백업 정책: https://policies.google.com/privacy

8. 사용자의 권리 · 옵트아웃

  • 데이터 삭제: 앱을 기기에서 삭제하면 모든 로컬 데이터가 함께 삭제됩니다.
  • 진단 데이터 옵트아웃: 안드로이드 기기 설정 → Google → 광고에서 "광고 ID 재설정" 또는 "광고 개인 맞춤설정 사용 중지"를 활용하거나, 앱을 삭제하면 됩니다.
  • 위치 권한 철회: 안드로이드 기기 설정 → 앱 → 안녕, 오늘 → 권한에서 위치 권한을 끄면 즉시 적용됩니다 (이 경우 현재 위치 날씨 기능이 제한됩니다).

9. 개인정보처리방침 변경

이 방침이 변경되는 경우, 본 페이지 상단의 "시행일"을 갱신하고 앱 업데이트 내역에 공지합니다.
중요한 변경(새로운 데이터 수집 등)이 있을 경우 앱 내에서도 별도 안내합니다.
